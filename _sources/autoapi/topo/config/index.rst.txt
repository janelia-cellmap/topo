topo.config
===========

.. py:module:: topo.config

.. autoapi-nested-parse::

   Default class configurations for flow generation and postprocessing.

   All postprocessing parameters are resolution-dependent, derived from
   GT instance statistics (see topo/scripts/gt_stats/ANALYSIS.md).

.. py:function:: get_instance_class_config(resolution_nm: int | None = None) -> dict[str, ClassConfig]

   Get flow generation config for a given resolution.

   :param resolution_nm: Resolution in nm (8, 16, 32, or 64).
                         Defaults to the module-level RESOLUTION variable.

   :returns: Dict mapping class name to flow generation parameters.

   :raises ValueError: If resolution_nm is None and RESOLUTION is not set.
